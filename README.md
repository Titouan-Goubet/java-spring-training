# React Spring Boot 17 Authentication API

Cette application est une API développée avec Spring Boot 17 pour gérer l'authentification des utilisateurs. Elle permet aux utilisateurs de s'inscrire, se connecter

## Fonctionnalités actuelles

- Inscription des utilisateurs avec validation des données.
- Connexion et gestion des sessions via des tokens JWT.
- Système sécurisé de gestion des mots de passe (hashage).

## Améliorations prévues

- Ajouter un système de récupération de mot de passe par email.
- Implémenter la gestion des rôles et des permissions utilisateurs.
- Ajouter une interface utilisateur plus poussée pour gérer les profils.

## Technologies utilisées

- **Backend :** Spring Boot 17 (Java)
  - Gestion des routes d'authentification (Spring Security).
  - JWT pour la gestion des sessions.
  - PostgreSQL/MySQL pour la gestion des utilisateurs (au choix).
